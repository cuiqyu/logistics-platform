在参考北京海关总署最近颁布的征税条例以及针对行李在运输途中可能出现的意外，我们为同学们详细制作了打包手册，以最大程度地保证行李的完整性并降低被税风险。

西游寄针对美国地区的同学免费邮寄一次打包材料（运单 运单保护袋 防水胶带），邮寄三箱以上免费赠送纸箱。由于疫情期间快递时效较慢，可以选择就近的UPS/FEDEX/HOMEDEPOT/LOWES门店购买纸箱，打印LABEL，之后后凭小票报销。

1） 建议购买Home Depot / Lowe's Heavy Duty Medium size的Moving box

2）UPS/FEDEX/USPS等门店购买的纸箱亦可，尽量购买带有HEAVY DUTY标识或者专门的MOVING BOX，购买的size以medium为宜 ，根据箱子上标识的尺寸可以自己计算体积重（计算公式：长* 宽*高/139<=50即满足要求）

1）国际行李的计费单位为磅（LB）而不是千克（KG）

2)国际快递的重量测量标准是：箱子的实际重量VS体积重量 取大值。重量大于50磅将有$24的超重费；大于70磅将产生$75的超重费；大于100磅需要走特殊渠道又更高额的费用。

3)纸箱上面标出来的尺寸为箱子内径， 箱子入库后将使用专业的光学扫描仪器去对整个箱子进行扫描测距，在每一个方向上会选取距离最远的两个点进行测量。 如果箱子装满以后，箱体向外鼓出，测量的距离将会是最鼓出来的两点之间的距离。

4）国际运输采用向上取整的方式

举个栗子：打包好纸箱后测量出來的长度是 18.3 inch, 于是将箱子进行挤压处理，认为四舍五入就是18 inch;实际入库之后，在机场行李专门的光学系统下量出來一定是 19 inch而不是 18 inch

由于中国海关对于行李物品大陆居民的免税额度是RMB,5000元 RMB，非大陆居民是2000元RMB

（请注意：5000元包括您本次回国时自身携带的物品，在机场海关处有记录）

因此我们对于有较多行李邮寄需求的同学提供一下建议：

1）在打包行李时准备一份行李清单给客服，以便后续工作人员可以尽快开展清关工作

2）混装：不同种类物品混装成一箱 （eg. 衣服/鞋子/保健品/包包），切忌不要同一种类的物品装一箱，例如整个箱子都是鞋或者包包，那么很大可能将会被重税

3）奢侈品包包建议一个MEDIUM SIZE的箱子不要装超过两个

4）全新或者高奢的物品很有可能在开箱查验时被重税或者送进缉私房，如果有相关邮寄需求的同学，需要提前告知客服，以便我们提早和海关沟通，尽量将税费控制在可承受范围并且保证行李的完整性。

5）易碎品需要用泡沫膜或者厚衣物包裹加固

6）禁运物品

7）纽约州以外的大部分地区需要自行打印LABEL贴在箱子上等待UPS/FEDEX/USPS工作人员上门取件。建议这些地区的同学将姓名和主单号用马克笔写在箱子外部以避免label脱落造成的行李遗失。

8）行李箱可以直接邮寄回国，但由于其材质特殊，用胶带很难固定面单，请尽量在FedEx，UPS，或USPS营业厅领取免费的pouchs and tags，将运单(label)放入邮袋(pouch)，将邮袋贴在吊牌(tag)上，最后将吊牌挂在行李箱把手上。

如果无法领取，可以用保鲜膜紧紧缠绕LABEL，并用多重胶带固定。
