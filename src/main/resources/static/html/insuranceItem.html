回国行李保险条款

一、保费金额

西游寄国际物流公司为客户免费赠送一箱子700美金，最高3000美金的保险服务；全美独此一家高额理赔；

二、适用范围

1. 行李是否已经进入运输环节，以西游寄官方物流信息为准。没有任何官方物流信息显示已进入运输环节的行李，不属于本保险条款理赔范围内。

2. 在运输环节中发生整件行李丢失的情况，客户需要向西游寄提供丢失物品的详细信息以及价值证明。西游寄会使用这些信息和联邦快递FedEx共同进行查找，在确认查找无果的情况下，按照单箱理赔$700,最高理赔$3500的原则向可进行理赔。

3. 行李送达目的地后收件人发现个别物品遗失以及包装破损，请第一时间联系西游寄在线客服，并提供具体丢失信息。西游寄在收到反馈详情后会立刻联系Fedex以及境内运输快递公司（顺丰&京东&德邦等）进行调查并申报理赔。

三、理赔流程

1. 西游寄在向客户收集所有物品丢失或损坏的证据后，向FedEx提交理赔申请。申请赔偿的具体流程与时间限制与FedEx的官方理赔流程一致，具体细节及具体保险条文请参考FedEx官方网站的相关信息（http://www.fedex.com/us/fcl/pckgenvlp/online-claims/，http://images.fedex.com/us/freight/pdf/Guide_to_Loss_and_Damage_Claims.pdf）

2. 客户只有在通过西游寄向FedEx或者UPS进行理赔的情况下，才能获得本《购买保险须知》条款中承诺的保险内容。客人如果绕开西游寄，直接与FedEx或UPS进行理赔手续，则被认为自动放弃通过西游寄进行相关理赔手续的权利，不再受本《购买保险须知》条款所保护，西游寄亦不对FedEx或UPS的理赔裁决负任何责任。

3. 理赔申请必须在行李丢失或损坏发生后三个自然日内向西游寄物流公司提出，超过该限期提出的理赔，不予受理.

4. 当行李发生损坏需要提出理赔时，客户需要对损坏的物品，包括外包装保存好作为证据，并且留下能够充分体现破损情况的照片存证。因收件人未能提供相关破损证据而导致理赔失败，西游寄不负相关责任。

国际个人包裹及奢侈品专线保险条款

一、保险价格

1. 西游寄向客户赠送运费三倍价值的免费保险。

2.客户亦可为包裹购买保险。保险费用为物品价值的10%。假设客户投保100美元，如果物品丢失，则享受1000美金的理赔。

二、适用范围

1. 在运输环节中发生的包裹丢失的情况，可获得的最高理赔金额为客户购买的保险金额

2. 客户在没有购买保险的情况下，如包裹发生破损或者丢失的情况，需要向西游寄提供丢失或者破损的物品的详细信息以及价值证明。在进行寻找和调查后，西游寄向客服进行理赔，最高赔偿金额不超过丢失物品运费的三倍。

三、理赔流程

1. 包裹在运输环节中发生包裹丢失或者破损的情况，客户需要在签收包裹的同时，向投递人员获取包裹包装破损证明并存证，然后将此证明连同包装箱全景正反面照片、包装箱上的快递面单照片、内件物品放在包装箱中的全景照片、拿出包装箱后的物品全景照片和破损物品的特写照片，在三个自然日内提供给西游寄物流公司。

2. 申请理赔时，无论是包裹丢失还是包裹损坏，客户都必须提供与订单详情中包裹内件信息一致的物品价值证明信息，否则理赔申请不予接受。

3. 包裹在运输过程当中，如在外包装无明显破损的情况下包裹内件发生破损，则可视为非运输环节产生的损坏，不属于该保险条款的赔偿范围之内。

4. 在收到客户的信息反馈后，西游寄物流会在第一时间进行调查理赔。自收到客户申请理赔五个工作日内完整调查取证，十个工作日内完成。
